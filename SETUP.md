# 仓库创建指南

你需要先手动创建仓库，步骤很快：

---

## 你来做（1 分钟）

1. 打开 [https://github.com/new](https://github.com/new)
2. 填写：
   - **Repository name**: `starx_vNext`
   - **Visibility**: `Private`
   - **不要勾选** "Add a README file"
   - **不要勾选** ".gitignore" 和 "license"
3. 点 **Create repository**

---

## 创建完成后告诉我

完成以上步骤后，请告知相关人员仓库已创建完成。
